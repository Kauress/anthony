Arrays

Definition

An array is a ordered list of elements, where each element has a value of any data type. You can define an array by putting a list of any values inside brackets ([]). To change a element in the array, you would start with the array name, then you would use the index of the item you want to change inside of brackets ([]). Then set the value equal to the new amount. You can add a element to the array by also starting with the array name, with the value set to the new element. Its gonna be added to the end of the array. You can also use the .push method to add an element to an array. First, you would start with the array name followed by the .push("") syntax for the method. Other than the push method you can also use the .concat() method. Like the push method you start with the name followed by the .concat().

If you wanted to remove an element from the array you could use the .pop() method. First, you would start with the array name followed by the .pop(""), and what you want removed in the paranthesis. The .splice("") method is a little similar to the pop method in the sense that it removes items. But it removes items and returns the removed items to a new array. With the forEach method you are able to iterate over a given array. It starts with the callback function and uses the elements value as a arguement. You can also perform equations inside of the arguement.

If you wanted to create a new array with a value that depends on a set array you can use the forEach method. The .filter("") method is a different way you can mutate the original array given by a arguement in the filter.

You can nest arrays inside arrays to create something kinda like sub-categories. Later on you can call in the onjects inside the nested arrays using the .index method. If you want to make your array look a little more clean you could use the .sort() method. This method sorts an array of unsorted numbers starting from the smallest to the biggest. The .reverse method turns around the order of the array.
