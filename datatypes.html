Javascript Data Types

Strings

Definition

A string is a set of characters that are placed inside quotation (single or double) marks. Strings can be assigned to variables, arrays, function, etc. A string can consist of sentences, quotes, numbers, etc. You can add a quote inside of a string as long as it doesnt match what around it. What is written inside of the string is what will be displayed onto the console or the webpage.

You can combine two seperate strings together to create one. This is called string concatenation. using the addition operator it will add the string on the left to the string on the right. Say you wanted to log a word in all upper-case. Well using the .toUpperCase method will log a woed or a whole string in all capital letters.

By using the .slice method, it will delete a item in a string in which you selected by using the .index method.You can also figure out the length (amount of characters) of a string using .length method. You can locate something inside of your string using the index method. Using this method allows you to select a specific string inside of a string. If you have an string with multiple strings inside of it, the first item in the string would be at index(0) and so on so forth. This is useful for selecting specific items in a string to change without changing the whole string.
