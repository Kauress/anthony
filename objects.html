Objects In Javascript

Objects

Definition

Objects in javascript can be thought of like real life objects. Each object has properties that are all unique to the object, and methods that help the object work.
Methods are actions that can be performed by the object. These methods are then stored inside of the property tag wich is paired with the object name. When a property is paired with the object name using a semi-colon, this is called a key value pair.

Excercises:

  1. Declare a variable called myHome.
    Inside of the myHome variable add as many properties that represent different values from your home.
    Then call on your variable to see all the different properties listed.
  2. Declare a variable called myCar.
    Inside of that variable add two properties of the car.
    Just ike the last excercise call on that variable to see different properties.
  3. Declare a variable called dog and set it equal to three different properties of your dog.
    Inside of the variable dog, create a function that logs the name of your dog.
  4. Declare a variable called room.
    Inside of the variable, give three properties of your room.
    Finally inside of the variable create a function called "myRoom" and in the function create an alert that alerts the properties of your room.
  5. Declare a variable called home.
    In that variable give five different properties of your dream home.
    Finally create an alert that alerts "This is my dream home".
