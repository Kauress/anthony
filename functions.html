Functions

Definition

Functions in javascript are a smaller program inside of javascript that performs a particular action.
The way you use a function that you already declared is by calling on it or better known as "invoking a function". When you call on a function, it willalways return a value even if you didnt give it a value.
If you didnt set a value to the function and call on it then it will return "undefined".

To define a function there are a couple of different ways you can do this. Either a function declaration or by function expressions.
Using the function declaration, it allows you to define a named function. To start off a function declaration you use the "function" keyword followed by what you want to name this function.
After the name of the function is followed by the parameters of this function. The parameters are set inside a set of parantheses

The function expressions are used to define a named function or a function that hasnt been named yet.
To define a function, parameters and arguements are used. There is a bit of similarities between the two but there is also a big difference.
Parameters are used to define the function. They are what defines the name of the function. An arguement is the actual value of the function name.
