Variables

Definition

A variable is like a container that stores data for a specific value. The variable must be named using a unique-name which is the identifier.

There are three steps for completing the process of a variable; Declaring, initalizing, and assigning:

  • Declaring a variable is giving the identifier to the variable tag. Meaning that there is no data stored in the value yet. In order to delcare a variale you have to be using one of three keywords. Var, let, or const. The let keyword and the var keyword both are used to declare a variable whose value can be changed later on down the road.
  • Assigning a variable is where you are giving the value or the identifier some data to store inside itself.
  • Lastly initalizing the variable is closing off the line of code so the computer is able to read it.
